# Clawd Desktop Pet Agent — Backlog (Post-MVP)

## P1 - 增强安全与智能化

### 主动建议
- 宠物在用户工作时检测上下文并主动提议帮助
- 示例：检测到大量代码变更 → "要帮你生成 commit message 吗？"
- 需要后台监听机制（文件系统 watch、进程检测等）
- 需要可配置的静默/打扰模式

### 操作预览与回滚
- Agent 执行前展示操作预览（将要做 X、Y、Z）
- 支持操作回滚（至少文件操作层面）
- 操作历史记录 + undo 栈

### 增强确认门
- 按风险等级分级：低风险（自动执行）/ 中风险（一次性确认）/ 高风险（逐步确认）
- 可配置的信任等级（用户可调整自动执行阈值）

## P2 - 工作流与场景

### 可视化工作流编辑器
- 拖拽式工作流编辑（类似 n8n / Zapier）
- 用户自定义场景模板（如"每天早上打开邮件 + 汇总到 Notion"）
- 工作流变量 + 条件分支

### 场景市场
- 社区共享工作流模板
- 场景评分/评论
- 一键导入

## P3 - 生态与扩展

### 插件系统
- 第三方 Tool 插件（类似 VSCode 扩展）
- 插件 API / SDK
- 插件市场

### 多宠物系统
- 不同宠物角色可选（不同 GIF 套装）
- 宠物换装 / 外观自定义
- 多宠物协作（不同宠物负责不同任务）

### 多设备同步
- 设置/任务跨设备同步
- 云端任务调度

## P4 - 交互增强

### 语音交互
- 语音输入指令
- 宠物语音回复（TTS）
- 语音唤醒词

### 移动端伴侣
- 手机端查看/管理任务
- 推送通知

### 多语言支持
- UI 国际化
- Agent 多语言理解（已天然支持，UI 层需要适配）
